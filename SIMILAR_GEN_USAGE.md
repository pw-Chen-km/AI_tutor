# Similar Generate 功能使用說明

## ⚠️ 重要：Similar Generate 目前只在 "In-Class Drills" 模組中可用

### 如何找到 Similar-Gen 按鈕

1. **切換到正確的模組**：
   - 點擊左側邊欄的 **"In-Class Drills"** （不是 Labs, Homework, 或 Exams）
   - 圖示：⚡ 閃電圖標

2. **生成題目**：
   - 上傳課程資料
   - 點擊 "Generate Drills" 按鈕

3. **找到 Variant Selector**：
   - 在每個題目卡片的 **CardHeader 底部**（Copy/Regenerate 按鈕下方）
   - 會看到一個分隔線，下方顯示：
   ```
   ─────────────────────────────────────
   Variants: [A ✓] [+ Similar] 
   ```

4. **生成類似題**：
   - 點擊 **"+ Similar"** 按鈕
   - 等待 LLM 生成類似題
   - 新變體會顯示為 [B], [C], [D]...

### 如果看不到按鈕

**檢查事項：**

1. ✅ 確認您在 **"In-Class Drills"** 模組（不是 Labs）
2. ✅ 確認已經生成了題目
3. ✅ 檢查瀏覽器控制台是否有錯誤
4. ✅ 嘗試刷新頁面 (F5 或 Ctrl+R)

### 截圖位置對照

您的截圖顯示：
```
Problem 5: Task 5: Resource Merging with Operator Overloading
```

這是 **Labs 模組**的題目格式！

Drills 模組的題目格式應該是：
```
Problem 1: [概念名稱]
Worth 5 points
```
